# Monsters Rolodex

**Version:** 0.1.0

Monsters Rolodex is a simple web application built with React for browsing a list of monsters. This project was created for learning and practicing React development.

## Features

- View a list of monsters with their names and images.
- Search for monsters by name.
- A simple, responsive user interface.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/monsters-rolodex.git

2. Navigate to the project folder:

   ```bash
   cd monsters-rolodex

3. Install the dependencies:
   npm install

4. Start the development server:
   npm start

5. Open your browser and go to http://localhost:3000 to see the app in action.
